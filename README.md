# Indian E-Commerce Pricing & Revenue Growth Analysis (36 Months)
Project Overview
This is an end-to-end Data Analysis project focused on analyzing Indian e-commerce pricing trends and revenue growth over a 36-month period.
The objective of this project was to:
* Clean and preprocess raw data using Python
* Store and analyze data using MySQL
* Solve business problems using SQL queries
* Build an interactive dashboard in Power BI
* Generate business insights from data

---
This project demonstrates practical skills in Data Analytics, SQL, and Business Intelligence.
* Tools & Technologies Used :
* Python (Pandas)
* MySQL
* Power BI

---
* Project Workflow

1. Data Processing in Python
* Loaded dataset using Pandas
* Handled missing values and duplicates
* Performed data type conversions
* Created calculated columns
* Conducted Exploratory Data Analysis (EDA)

2. Data Storage & Business Analysis in MySQL
* Imported cleaned dataset into MySQL
* Wrote SQL queries to solve 10 business questions
* Used GROUP BY, ORDER BY, aggregate functions, joins, and subqueries
* Analyzed revenue growth, category performance, and monthly trends

3. Power BI Dashboard
* Connected Power BI to MySQL database
* Created an interactive dashboard
* Designed visualizations for:

  * Monthly Revenue Trends
  * Category-wise Performance
  * Pricing Analysis
  * Growth KPIs
  * Added slicers and filters for dynamic analysis

---

*Key Business Insights

* Identified high-performing product categories
* Analyzed seasonal revenue growth patterns
* Observed pricing impact on revenue
* Generated actionable business insights for decision-making
